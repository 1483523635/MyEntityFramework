这个框架简单实现了EF的功能 规则和EF一致
有GetAllInfo（）//获取所有的信息
Add（）//添加一条记录到数据库中
Remove()//删除一条记录
Update（）//更新一条记录


环境配置 将该目录下的appsettings.json 文件复制到你的项目的根目录中 
         修改服务器的地址，和数据库名称。



测试简单样例
  
  Users u = new Users() { ID = 1018, Name = "name_new" ,pwd="123" };
  BasicsDatabase _Context = new SqlServerDatabase();
  _Context.Add(u);      //执行成功返回true 不成功返回false
  _Context.Update(u);   //执行成功返回true 不成功返回false
  _Context.Remove(u);   //执行成功返回true 不成功返回false
  var list = _Context.GetAllInfo<Users>();
